Entender a economia é vital porque recursos como RR (estilhaços de reroll) e Gemas servem a propósitos diferentes. Enquanto as Gemas servem para invocações, os estilhaços de RR são críticos para otimizar os atributos das suas unidades.

| Recurso | Uso Principal | Prioridade de Aquisição |
| :--- | :--- | :--- |
| **Gemas** | Invocar novas unidades | Alta |
| **RR (Reroll)** | Alterar atributos de unidades | Média |
| **Cubos** | Materiais de melhoria | Alta |
| **Malícia** | Evolução/Habilidade | Média |

<Callout type="tip" title="Gerenciamento de Recursos">
Sempre mantenha uma reserva de gemas para banners de tempo limitado. Gastar toda a sua moeda em banners padrão geralmente faz com que você perca unidades Secretas que definem o meta.
</Callout>
